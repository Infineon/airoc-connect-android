CySmart Android App Open Source Libraries
-----------------------------------------

CySmart Android app uses few open source libraries to implement some of its features. 
The details of these open source libraries are documented below:

1. AChartEngine
AChartEngine is a charting library for Android applications
https://code.google.com/p/achartengine/

License: Apache License2.0
https://github.com/ddanny/achartengine/blob/master/LICENSE-2.0.txt

2. Android-WebRequest
Fluent interface for easy HTTP requests to web servers (GET, POST, PUT or DELETE)
https://github.com/delight-im/Android-WebRequest

License: Apache License 2.0
https://github.com/delight-im/Android-WebRequest/blob/master/LICENSE

3. Layout files (layout_menu_paircache.xml, listitem_device.xml, listitem_firmware.xml, listitem_glucose_spinner.xml, simple_spinner_dropdown_item.xml, simple_spinner_item.xml)
Represent the XML based views.

License: Apache License 2.0
http://www.apache.org/licenses/LICENSE-2.0

4. GlucoseContextRecord.java, LineGraphView.java
GlucoseContextRecord.java represents a Glucose BLE profile data package.
LineGraphView is a line graph chart control.
